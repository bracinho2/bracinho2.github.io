Player Master Sul

O projeto "Player DMR Master Sul" surgiu da oportunidade de ter disponível um aplicativo para escutar os TGs 724 e 72441 em uma interface amigável programada para dispositivos móveis, especialmente celulares.

Isto foi possível porque a Rede Master Sul disponibiliza os fluxos de audio diretamente em seu site, logo, foi possível aproveitar os fluxos e desenvolver um aplicativo com uma função apenas: escutar os radioamadores que ali frequentam.

O aplicativo foi escrito em Dart/Flutter e está disponível para download. As atualizações são feitas de acordo como tempo/habilidade do desenvolvedor. Portanto, o aplicativo se trata de um "teste" contínuo.

Versões:

01/01/2021 > Versão 1.2

Esta versão do Player inclui o Player para o TG 724 e TG 72441. Incluimos também um questionário para que os radioescutas possam estudar e responder as questões da Anatel para realizarem as provas para se tornarem radioamadores.

Player DMR Master Sul - Versão 1.2

Repetidoras

Existem muitas opções de aplicativos que exibem as repetidoras disponíveis para os radioamadores. Muitos deles são internacionais e não representam a realidade do Brasil, outros foram desenvolvidos com tecnologias que impossibilitam um acesso adequado em dispositivos móveis.

A ideia geral deste Aplicativo é termos uma lista construida coletivamente por todos os radioamadores com interface amigável para dispositivos móveis. As atualizações são feitas de acordo com o tempo/habilidade do desenvolvedor. Portanto, o aplicativo se trata de um "teste" contínuo.

Versões

01/01/2021 > Versão 1.0

Versão lançamento com a exibição das repetidoras informadas e possibilidade de inclusão de novas repetidoras diretamente no Aplicativo. Lembrem-se sempre de enviar um 73 para os radioamadores e um 88 respeitoso às radiamadoras!
